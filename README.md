Interactive visualizations and computational explorations of differential geometry concepts.

## About

Coursework for **Curvas e Superficies** (Curves and Surfaces) at EMAp/FGV, taught by Professor Maria Soledad, PhD, and Professor Asla Sa, PhD (2017.2).

This repository contains GeoGebra constructions, Desmos visualizations, LaTeX write-ups, and Python/MATLAB scripts that bring classical differential geometry to life through interactive and animated examples.

## Topics Covered

- **Bezier Curves** -- construction and animation of 2D and 3D Bezier curves
- **Roulettes** -- cycloids, epicycloids, and hypocycloids
- **Parametric Curves** -- tangent lines, tangent vectors, and singular points
- **Classical Curves** -- astroid, logarithmic spiral, Caracol de Pascal, helix
- **Surfaces** -- Viviani's curve, Whitney umbrella, helicoid, bivalve shells, surfaces of revolution
- **Curvature and Torsion** -- evolutes, osculating circles

## Tech Stack

- **GeoGebra 5.0** -- interactive geometry constructions
- **Desmos** -- parametric curve visualizations (exported as HTML)
- **LaTeX** -- formal write-ups and reports
- **Python** -- helix generation script
- **MATLAB** -- surface plotting scripts (helicoid, bivalve, Whitney umbrella, Viviani's curve)

## Repository Structure

```
homework-1/       Plane curve exercises (GeoGebra)

project-1/        Bezier curves and helix (GeoGebra, HTML, Python)

```
